# Logo Placeholder

Questo file è un placeholder. Per utilizzare il tuo logo:

1. Salva il file del logo come `logo.png` o `logo.svg` in questa cartella
2. Assicurati che il file abbia dimensioni appropriate (consigliato: larghezza di circa 200-300px)
3. Se il logo ha uno sfondo trasparente, il formato PNG o SVG è preferibile

## Istruzioni per l'implementazione

Una volta salvato il logo in questa cartella, potrai accedervi tramite il percorso `/images/logo.png` o `/images/logo.svg` nel codice.
