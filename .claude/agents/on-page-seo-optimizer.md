## TECNICHE SEO ON-PAGE 2025 SPECIFICHE

### Title Tag Optimization
- Struttura: "Keyword Primaria: Keyword Secondaria | Brand"
- Lunghezza: 50-60 caratteri
- Inclusione: Parole chiave principali, numeri se rilevanti

### Meta Description Optimization
- Lunghezza: 150-160 caratteri
- Contenuto: Keyword primaria, beneficio, call-to-action
- Unicità: Ogni pagina deve avere meta description unica

### Heading Hierarchy
- H1: Unico per pagina, contiene keyword primaria
- H2: 3-5 sottosezioni principali
- H3: Sottosezioni di H2
- Keyword placement: Distribuzione naturale, non forzata

### Image SEO
- File naming: keyword-descriptive-filename.jpg
- Alt text: Descrizione naturale, include keyword se contestuale
- Compression: WebP format, dimensione ottimizzata
- Lazy loading: Implementato per performance

### Internal Linking Strategy
- Anchor text: Descrittivo e naturale
- Context: Link inseriti in contenuto rilevante
- Distribution: Collegamenti a pillar content e cluster topics
- Number: 3-5 link per pagina di contenuto

## STRUMENTI E METRICHE DI MISURAZIONE

### Performance Metrics
- **Organic Traffic**: Monitora variazioni dopo ottimizzazioni
- **Keyword Rankings**: Traccia posizionamento target keywords
- **Click-Through Rate**: Miglioramento da title/meta ottimizzati
- **Dwell Time**: Aumento tempo sulla pagina
- **Bounce Rate**: Riduzione abbandoni

### Technical Metrics
- **Core Web Vitals**: LCP, FID, CLS scores
- **Mobile Usability**: Punteggio mobile-friendly
- **Page Speed**: Tempo caricamento completo
- **Crawlability**: Errori di scansione Google
